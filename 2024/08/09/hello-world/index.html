使用 Hexo 创建博客

2024-08-09 · 2 min · 470 words · 博客

Hexo 是一款快速简介高效的博客框架， 支持 markdown 格式， 支持主题， 支持插件， 支持一键部署到 GitHub Pages。

Quick Start

准备工作

需提前安装 Node.js 和 Git。

全局安装 Hexo

使用 npm 安装 hexo-cli 和 hexo-deploy-git。

  • hexo-cli：Hexo 的命令行工具
  • hexo-deploy-git：Hexo 的 Git 部署插件

创建博客项目

创建 Hexo 项目

安装依赖

本地预览

配置 Hexo

通过博客项目的 _config.yml 文件配置博客信息。

主题配置

在 Hexo 官网上有丰富的主题，可以参考 主题。

选择自己喜欢的主题后，通过主题 Github 仓库下载主题，将主题文件复制到博客项目的 themes 目录下。

然后在 config.yml 文件中配置主题。

然后重新构建，启动本地服务既可在本地预览

部署到 GitHub Pages

  1. 在 GitHub 上创建一个仓库，仓库名称为 <username>.github.io，例如 johndoe.github.io。

  2. 在博客项目的 _config.yml 文件中配置部署信息。

  1. 构建并部署。
  1. 访问 https://<username>.github.io，即可看到博客。

添加文章

  1. 在博客项目的 source/_posts 目录下新建 markdown 文件，例如 hello-world.md。

  2. 在 markdown 文件中添加文章内容。
